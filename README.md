## 🏗️ Featured Projects

### 🔵 AlgoAdda — Full-Stack Coding Platform
> A LeetCode-style competitive coding platform with problem management, user authentication, and a rich code editor.

- **Stack:** React, Redux, Node.js, Express, MongoDB, HTTP-only Cookies
- **Deployed on:** Vercel (frontend) + Render (backend)
- **Highlights:**
  - Cookie-based auth with `SameSite=None; Secure` for cross-origin sessions
  - CORS configured for production Vercel URL
  - Protected routes with Redux-managed session state
  - Axios client configured with `withCredentials: true`
